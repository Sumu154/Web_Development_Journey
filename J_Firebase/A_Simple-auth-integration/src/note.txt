steps of firebase

1. go to firebase.google.com and then go to console
2. create a firebase project
3. go to web settings for firebase app and register the app
4. npm install firebase in vs code
5. get firebase config and [you should not] add the config in your project
6. enable sign in method you want to use: 
Build > Authentication > Get Started > click on google(or whatever you want to add) >
Enable > Set the email address > Save
7. go to firebase docs
8. --->
